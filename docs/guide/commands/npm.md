# NPM

## 説明

[npmjs](https://npmjs.com)に公開されている。パッケージの情報を表示したり検索します。

## 使用法

`yui!npm <search|registry> <query:...string>`

### 検索

`yui!npm search <検索したいパッケージの名前>`

![Imgur](https://i.imgur.com/PVasmwo.png)

### 指定したパッケージの詳しい情報を確認する。

`yui!npm registry <パッケージの名前>`
