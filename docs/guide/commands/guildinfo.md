# GuildInfo

## 説明

サーバーの情報を表示します。

## 使用法

`yui!《guildinfo|serverinfo》`

## 詳しい説明

![GuildInfo](https://i.imgur.com/QYQL6GZ.png)

### タイトルの部分

ここにはサーバーの名前が表示されます。

### Guild ID

ここにはサーバーのIDが表示されます。

### Owner

サーバー所有者のタグと名前が表示されます。

### Members

サーバーにいるメンバーの数が表示されます。  
ボットの数やオフラインの数も確認できます。

### Channels

カテゴリ、テキストチャンネル、ボイスチャンネルの数を確認できます。

### Emoji

サーバー絵文字の数を確認できます。

### Roles

役職の数と名前を確認できます。

### Created At

サーバー作成日が表示されます。
