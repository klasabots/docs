# Conf

## 説明

サーバーで使える機能の設定ができます。

## 使用法

`yui!conf <set|show|remove|reset> (key:key) (value:value)`

### Set

`yui!conf set (キーの名前) (値)`  
これでキーの部分に値を設定する事ができます。

### Show

`yui!conf show (キー)`  
これでキーと値の部分になにがあるか確認する事ができます。

### Reset

`yui!conf reset (キー)`  
これで値の部分を初期値に戻す事ができます。

### Remove <Badge text="Setと動作が同じ" type="warn"/>

`yui!conf remove (キー) (値)`
