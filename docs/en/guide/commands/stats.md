# Stats

## Description

Display bot details and statistics.

## how to use

`yui!stats` 
 When you send it, a message like the one below will be returned.

```asciidoc
= STATISTICS =

• Mem Usage  :: 85.13 MB
• Uptime     :: a day
• Users      :: 33,134
• Guilds     :: 45
• Channels   :: 1,219
• Klasa      :: v0.5.0-dev
• Discord.js :: v12.0.0-dev
• Node.js    :: v10.15.3
• Shard      :: 1 / 1
```
